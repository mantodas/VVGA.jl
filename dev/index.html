Installation

Users

  1. Download Julia v1.6.4 or later, if you haven't already.
  2. Add the LTVsystems module entering the following at the REPL ]add https://github.com/mantodas/VVGA.jl.

Student Developers

  1. Clone the LTVsystems module to username/.julia/dev/.
  2. Enter the package manager in REPL by pressing ] then add the package by typing dev VVGA rather than add VVGA.
